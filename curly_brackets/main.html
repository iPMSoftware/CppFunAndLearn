Curly brackets initialization

When code from main.cpp is compiled with gcc 4.9.4 and C++14 result of it is presented below:

#include int main() { auto x {1}; std::cout << "x jest typem int: " << std::boolalpha << std::is_same::value << std::endl; std::cout << "x jest typem std::initializer_list: " << std::boolalpha << std::is_same>::value << std::endl; }

Output is following:

x jest typem int: false x jest typem std::initializer_list: true

But when compiled with GCC 5.5.0, output is different:

x jest typem int: true x jest typem std::initializer_list: false
